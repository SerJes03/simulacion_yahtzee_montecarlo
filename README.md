# 🎲 Simulación de Yahtzee con Método Monte Carlo

## Descripción

Este proyecto implementa una simulación del juego **Yahtzee** utilizando el método de **Monte Carlo**, con el objetivo de analizar probabilidades y resultados posibles a partir de múltiples ejecuciones aleatorias.

## 🎯 Objetivo

Aplicar técnicas de simulación para:

* Evaluar estrategias de juego
* Analizar probabilidades de combinaciones
* Comprender el comportamiento del azar en el juego

## Tecnologías utilizadas

* Python
* Jupyter Notebook
* Librerías:
  * random
  * pandas
  * matplotlib
  * seaborn

## ▶️ Cómo ejecutar el proyecto

### Opción 1: Google Colab

1. Haz clic en el botón **"Open in Colab"** dentro del notebook
2. Ejecuta la celda

### Opción 2: Local

1. Instala Jupyter Notebook
2. Abre el archivo:
   Simulacion_Yahtzee.ipynb
3. Ejecuta las celdas

## 📊 Resultados esperados

El notebook permite visualizar:
* Distribuciones de resultados
* Frecuencia de combinaciones
* Análisis estadístico de la simulación
